# About the Bluesky Feed Digest

For developers, enthusiasts, and anyone interested in the ATPROTO protocol and Bluesky development, here is a curated digest of the :rocket: ATPROTO Dev feed :computer: featuring updates, discussions, tools, and resources for the protocol and its ecosystem.

## What's included

- **DEVELOPMENT FOCUS** — protocol updates, tools, libraries, developer discussions
- Posts are collected weekly from the [ATPROTO Dev feed on Bluesky](https://bsky.app/profile/did:plc:atproto.dev/feed/dev){:target="_blank"}

## How it works

This site is automatically generated every week by a GitHub Actions workflow:

1. A Python script fetches the latest posts from the Bluesky public API
2. Posts are filtered by date and formatted into Markdown
3. Mkdocks Material ([Zensical](https://zensical.org){:target="_blank"} doesn't support the blog plugin yet) builds the static site
4. The site is deployed to GitHub Pages

The source code is open — feel free to adapt it for your own Bluesky feed digest!
