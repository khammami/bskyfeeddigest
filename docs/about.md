# About the Global Ecology Feed Digest

For the lazy (yes we are) and friends who do not like social media (yes they can) but could benefit from the news on the Global Ecology feed... here is a curated digest of the :butterfly: Bluesky Global Ecology feed :earth_africa: on biodiversity, ecosystems & conservation at large scales. Terrestrial, freshwater & marine realms.

## How it works

This site is automatically generated every week by a GitHub Actions workflow:

1. A Python script fetches the latest posts from the Bluesky public API
2. Posts are filtered by date and formatted into Markdown
3. [Zensical](https://zensical.org){:target="_blank"} builds the static site
4. The site is deployed to GitHub Pages

The source code is open — feel free to adapt it for your own Bluesky feed digest!
